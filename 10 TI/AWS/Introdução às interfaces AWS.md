3 maneiras de trabalhar com a AWS:
- Console de gerenciamento
- Linha de comando - CLI
	- Permite customizar e automatizar o uso
- Software development kits (SDKs)
	- Permite customizar e automatizar o uso

Atenção que pode haver um delay para visibilidade de alterações entre essas interfaces. uma alteração via CLI pode não aparecer imediatamente na console.
