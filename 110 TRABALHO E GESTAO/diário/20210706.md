#### Trabalho Serpro no dia
```dataview
TABLE file.mtime FROM "Serpro2021"
WHERE file.mtime >= date("2021-07-06") and file.mtime < (date("2021-07-06") + dur(1 day)) 
SORT file.mtime asc
```

#### Trabalho não Serpro no dia
```dataview
TABLE file.mtime FROM -"Serpro2021" 
WHERE file.mtime >=  date("2021-07-06") and file.mtime < (date("2021-07-06") + dur(1 day))
SORT file.mtime asc
```
