#### Trabalho Serpro no dia
```dataview
TABLE file.mtime FROM "Serpro2021"
WHERE file.mtime >= date("2021-09-10") and file.mtime < (date("2021-09-10") + dur(1 day)) 
SORT file.mtime asc
```

#### Trabalho não Serpro no dia
```dataview
TABLE file.mtime FROM -"Serpro2021" 
WHERE file.mtime >=  date("2021-09-10") and file.mtime < (date("2021-09-10") + dur(1 day))
SORT file.mtime asc
```
