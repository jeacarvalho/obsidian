# Diferencial
- Queremos ser revolucionários. Queremos mudar o mundo. Mas entendemos que o mundo só muda se o ser humano mudar. E só é possível mudar o ser humano pela conversão, pelo novo nascimento. Assim, só o evangelho muda o mundo porque só o evangelho muda o ser humano.
- Gente só muda com gente. Temos que ser relacionais. Ser aberto ao outro e honrar a abertura dos outros a mim.
- Queremos nos voltar para fora, não para dentro. Assim, a maioria esmagadora dos recursos financeiros deve ser para fora e não para autoalimentar nossa própria estrutura.
