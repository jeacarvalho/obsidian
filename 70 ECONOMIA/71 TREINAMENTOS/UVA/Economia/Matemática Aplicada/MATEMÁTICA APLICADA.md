## CONTEÚDO 1 Introdução ao estudo das funções

### [O que é modelagem matemática?](https://player.vimeo.com/video/340424992)
- Toda organização tem processos, sejam produtivos, sejam organizacionais
- Modelar esses processos é alvo do estudo da modelagem matemática
- O que é uma função? É algo que leva valores de um domínio (conjunto de partida) a uma imagem (conjunto de chegada). Custo, por exemplo, é uma função entre quantidade de matéria prima e produtos acabados. Uma função relaciona variáveis
- Função do 1o grau envolve uma variável e uma potência de 1; De 2o grau tem uma potência de 2. No fim, temos uma função polinomial de ordem "n"
- Função racional: razão entre dois polinômios
- FUnção exponencial: caso do Juros compostos, valor futuro
