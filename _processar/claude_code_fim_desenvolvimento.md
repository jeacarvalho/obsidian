# Claude Code e o fim do desenvolvimento como conhecemos

> Autor: Marco Souza (oprogramador) - Programador há mais de 20 anos

## Resumo

O **Claude Code** é uma ferramenta de IA que está transformando o desenvolvimento de software. O autor compartilha 4 dicas essenciais para usar a ferramenta de forma efetiva.

## 4 Dicas Principais

### 1. Comece sempre com o modo "plan"

- No modo plan, o Claude Code é somente leitura
- Ele busca informações e apresenta um plano de ação
- Não confunda no plano inicial - sempre falta algo
- Mantenha a conversa em inglês desde o início

### 2. Use arquivos CLAUDE.md com "divulgação progressiva"

- Arquivo CLAUDE.md na raiz do projeto
- Use comando `/init` para criar automaticamente
- Evite arquivos muito grandes
- Use "divulgação progressiva" - informe onde buscar info adicional
- Referência: [claude-code-bootstrap](https://github.com/oprogramadorreal/claude-code-bootstrap)

### 3. Testes de unidade e zelo pela qualidade do código

- Claude Code roda testes e se autocorrige
- Alta cobertura de testes é essencial
- Código legível ajuda a IA a performar melhor
- Aplique princípios: DRY, SOLID, KISS, YAGNI, clean code

### 4. Não escreva mais código na mão

- IA preenche "gaps" de conhecimento
- Permite usar tecnologias que antes não cogitaria
- Mais tempo para criar e menos para sintaxe

## Observações do Autor

- Custo: O autor paga $200/mês pelo plano profissional
- Alternativas: OpenCode, Ollama (local)
- O "pay to code" é uma preocupação
